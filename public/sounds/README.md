# 🔊 게임 효과음 파일

이 폴더에 다음 3개의 음원 파일을 추가해주세요:

## 필요한 파일
1. **correct.mp3** - 정답 효과음 (밝고 긍정적인 소리)
2. **wrong.mp3** - 오답 효과음 (부정적인 소리, 하드모드 전용)
3. **click.mp3** - 버튼 클릭 효과음 (짧고 가벼운 클릭음)

## 음원 파일 준비 방법

### 옵션 1: 무료 음원 사이트에서 다운로드
추천 사이트:
- **Freesound.org** - https://freesound.org/
- **Mixkit** - https://mixkit.co/free-sound-effects/
- **Zapsplat** - https://www.zapsplat.com/ (무료 회원가입 필요)
- **Pixabay** - https://pixabay.com/sound-effects/

### 옵션 2: AI로 생성
- **ElevenLabs Sound Effects** - https://elevenlabs.io/sound-effects
- **AudioGen** 등

### 검색 키워드
- **correct.mp3**: "success", "correct", "win", "positive", "ding"
- **wrong.mp3**: "error", "wrong", "fail", "buzzer", "negative"
- **click.mp3**: "click", "tap", "button", "ui click"

## 파일 형식
- MP3 형식 권장 (모든 브라우저 호환)
- WAV, OGG도 가능
- 파일 크기: 100KB 이하 권장 (빠른 로딩)
- 길이: 0.5~2초 권장

## 파일 배치 후
음원 파일을 이 폴더에 넣으면 게임이 자동으로 인식합니다.
별도의 설정 변경 없이 바로 작동합니다!
